#知了停车

## 介绍
知了停车为停车场和用户搭建的智能化停车场信息网络平台，融合了多种先进技术，帮助建立良好的城市交通环境，实现城市“智慧停车”。

##知了停车有哪些主要功能
知了停车的主要服务是提供**停车位的预测服务**。借助于深度学习算法，对目标停车场的车位数进行实时预测，为用户的停车提供参考，基于预测的服务，我们也开发了相关快应用导航，web端管理，车位推荐等相关功能，为用户提供一套从确定目的地到缴费离开停车场的服务系统

###深度学习算法简介

####车辆识别-YOLO-v3
2018 年，YOLOv3 被提出，在 YOLOv2 的基础上做出了一些 细节上的改进，并采用了更复杂的网络结构，参考 Res Net 的思想在卷积网络中引 入了残差块，同时提出了多尺度特征融合的方法，在提高了检测精度的同时保持了 高检测速度，并且提升了对小目标的检测效果。以下是 YOLOv3 的网络结构：
![](https://ftp.bmp.ovh/imgs/2020/04/454e9efb4d8b9048.png)
![](https://ftp.bmp.ovh/imgs/2020/04/636be7c4ca0a53f6.png)

###泊车预测- Bi - LSTM 算法
LSTM 算法突出特点是可以学习时间序列的长期依 赖信息，可以较好地描述复杂非线性问题，能够从数据样本中自动学习以前的经验 而无需反复查询和表现过程。Bi - LSTM 算法对LSTM网络预测后的时间序列再进一步进行学习训练，以有效克服预测误差大的缺点，具有捕获数据时序性和长期依赖性的优势
![](https://ftp.bmp.ovh/imgs/2020/04/a850979b41467e2d.png)

### Summary - 概要
知了停车是基于车位预测服务建立的一套停车辅助系统，旨在方便用户的开车出行，为用户的停车提供便利，解决停车难的问题
### Features - 特性
* 知了停车，用户可通过安装知了停车快应用，享受其带来的便利服务。
* 停车场管理放，借助摄像头+AI+web，实时了解停车场内部情况，管理更高效。与市面上现有的停车服务软件，我们的优点在以下几个方面

+ 用户
   * 充分利用了快应用的优势，减少下载安装的时间。即点即用，并且享受原生应用的性能。
   * 操作简单，无需学习轻松上手
+ 停车场管理方
   * 物联网的实现，在现有摄像头的基础上改装即可，减少安装地磁的成本
   * AI参与停车场的管理，管理更高效，充分利用停车场的资源

## Installation - 安装
+ 安装 NodeJS 需安装8.0以上版本的 NodeJS (建议使用 10.0+ 以上)
+ 手机安装调试器 调试器是一个 Android 应用程序，下载调试器 APK
+ 手机安装平台预览版，平台预览版是一个 Android 应用程序
+ 安装 toolkit，hap-toolkit 为开发编译工具

## Usage - 用法
